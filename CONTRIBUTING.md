# Contribute to Mrapi

## Development

1. Folk

   Folk the repository to your own account.

2. Start development

   ```bash
   yarn
   yarn build
   yarn dev
   ```

## Run tests

```bash
yarn test
```

## Git Commit

1. Add commit plugin

   ```bash
   npx fbi add factory-commands
   ```

2. Commit

   ```bash
   fbi commit
   ```
